git 管理项目的基本操作

一、管理项目常用命令介绍：

1、git version 查看版本信息， git help 帮助命令， git init 初始化仓库， git add filename 提交单个文件， git log 查看提交的版本，

2、git add . 将所有文件和文件夹提交到待提交文件的队列中

3、git commit -m "add some files..." , -m "" 添加提交到版本库的版本描述信息

二、cmd展示如何将本地项目备份到云端：

(1)注册登录云端git服务器

(2)在云端创建一个备份项目

(3)克隆云端项目到本地 git clone http

(4)复制本地项目到git项目中

(5)cd 项目的目录

(6)提交所有文件和文件夹 git add .

(7)git commit -m "描述" 提交到版本库

(8)git push origin master 提交到云端版本库
